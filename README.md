<h1 style="text-align: center">坤坤博客系统</h1>

## 项目简介
为了安利我们家鸽鸽，我们开发出了一个博客系统，这个系统可以聊天，可以发帖子交流。当然了，发布的内容与我们家鸽鸽无关的话，会提示非法内容而无法发布哦。所以请各位真爱粉们为坤坤博客贡献一份力量吧。

## 技术选型
### 1、系统环境
- Java EE 8
- Servlet 3.0
- Apache Maven 3

### 2、主框架

- Spring Boot 2.6.3
- Redisson 3.16.0

### 3、持久层

- Apache Mybatis-plus 3.5.1
